
# Triplit - Explore India - Travel Discovery Web App

## Project Overview
Explore India is a dynamic web-based travel discovery platform built using Flask and MySQL. 
It allows users to explore tourist locations across different Indian states.
You can make your excellent trip.

## Features
- API-based dynamic location fetching
- Server-side pagination
- Categorized places (heritage, nature, religious, beach, etc.)
- Image rendering using Wikimedia URLs
- Clean MySQL database structure

## Tech Stack
- Backend: Python (Flask)
- Frontend: HTML, CSS, JavaScript
- Database: MySQL
- API: REST Architecture

## Future Scope
- User authentication
- Booking system
- Favorites system
